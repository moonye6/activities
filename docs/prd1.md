核心功能拆解（优化版）
1. 活动创建
一键发布：用户只需填写关键字段（如时间），系统根据标签和兴趣自动匹配潜在参与者
自定义模板：保存常用活动模板（约吃饭、运动、聚会等），快速复用
活动信息设置：
时间、地点（可选）、活动类型
公共信息：谁能看到（可见范围）
标签选择：
系统标签：公司、部门、校友等
个人标签：自定义兴趣或社交圈子
参与人数限制：
单人邀请（定向约）
多人开放（系统自动匹配/筛选）
2. 私密性与可见性控制
可见范围选择：
朋友可见
特定标签可见
公共/半公开（比如同公司、同兴趣组）
隐私保护：
不公开敏感信息（位置可模糊）
可选择匿名或仅昵称显示
3. 邀请与匹配
精准匹配：
系统自动筛选符合条件的人（时间、标签、兴趣）
支持主动邀请（选定单人或多人）
互动确认：
参加/拒绝/暂定状态
自动统计参与人数，实时更新活动状态
4. 智能推荐
地点/活动推荐：
餐厅、场馆推荐（根据人数、评分、位置）
可基于历史活动偏好进行推荐
潜在参与者推荐：
系统根据标签、兴趣、活跃度推荐可能参与的人
5. 提醒与跟进
活动前提醒（可推送或小程序内提醒）
活动结束后反馈：
可分享照片、心得
参与者互评（增加信任机制）
6. 高级功能（可扩展）
活动类型扩展：电影、运动、课程、聚会等
兴趣/社交圈管理：可管理标签组、关注兴趣圈
数据分析：用户可查看活动参与率、好友活跃度
付费增值：
高级匹配（更多参与者推荐）
高品质场地/餐厅推荐
优先曝光活动

💡 总结核心价值：

极简创建 → 减少沟通成本
标签与私密性 → 精准匹配
自动统计与推荐 → 提高活动成功率
